# 2026 Timeline Tracker

A motivational dashboard to track your progress through 2026.

## Features
- **Year Tracker**: Visual progress of the year 2026.
- **Month Tracker**: Visual progress of the current month.
- **Target Tracker**: Countdown/Progress to May 3rd, 2026.
- **Motivational Mantras**: Context-specific affirmations to keep you focused.

## Deployment on GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Source**, select `main` branch (or `master`) and `/ (root)` folder.
4. Click **Save**.
5. Your site will be live at `https://<your-username>.github.io/timeline-tracker/`.

## Customization
- Edit `script.js` to change the target date or mantras.
- Edit `style.css` to change the color theme.
